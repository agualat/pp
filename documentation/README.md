## 📝 Contribuir a la Documentación

### Al agregar nueva documentación:

1. **Ubicación:**
   - Guías generales → `documentation/`
   - Docs de componentes → Dentro del componente (`client/`, `server/`, etc.)
   - Docs de scripts → `scripts/README.md`

2. **Formato:**
   - Usar Markdown
   - Incluir tabla de contenidos para docs largos
   - Agregar ejemplos de código cuando sea relevante
   - Usar emojis para mejor navegación visual

3. **Actualizar índices:**
   - Actualizar este README
   - Actualizar README principal si es relevante
   - Actualizar CHANGELOG.md

4. **Estilo:**
   - Claro y conciso
   - Incluir comandos de ejemplo
   - Mencionar casos de uso comunes
   - Incluir troubleshooting cuando aplique
